# lightmetro

> Nuxt.js project

## Build Setup

```bash
# install dependencies
# need to yarn install
$ yarn

# serve with hot reload at localhost:3000/lightmetro
$ yarn run dev

# deploy to github
$ yarn run generate
$ yarn run deploy
```

For detailed explanation on how things work, checkout the [Nuxt.js docs](https://github.com/nuxt/nuxt.js).
